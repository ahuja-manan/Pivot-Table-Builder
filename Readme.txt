Steps to run our web application


1) Upload the phase3.zip file to IVLE. Don’t forget to unzip it.


For stage 1
1) To run the form, open the pivot_table.html file and serve it.
2) Select the dimensions, values and filter values and click submit to display the table.
IMPORTANT: Filter values should only be separated by commas and not whitespaces or any other characters.


For stage 2
1) For visualisation, serve the visualisation.html file.
2) Click on a question to display the graph that answers it.
